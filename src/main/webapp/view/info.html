Wallet Control é um programa que controla lançamentos futuros, contas em atraso, contas pagas e gastos diversos, podendo ser filtrado por categoria, prioridade ou data, dando ao usuário um melhor controle de suas despesas mensais.

Tecnologias usadas
  • MySQL - MySQL Workbench 8.0 CE
  • Java - Eclipse
  • Maven
  • Jackson
  • Jersey
  • Json
  • Html - Bootstap Studio
  • JavaScript - jQuery - AngularJS
  • CSS
