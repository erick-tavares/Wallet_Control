
# Wallet Control

<h4>Wallet Control é um programa que controla lançamentos futuros, contas em atraso, contas pagas e gastos diversos, podendo ser filtrado por categoria, prioridade ou data, dando ao usuário o melhor controle de suas despesas mensais.</h4>

**_Tecnologias usadas_**
<ul>
<li>MySQL - MySQL Workbench 8.0 CE</li>
<li>Java - Eclipse </li>
<li>Maven</li>
<li>Jackson</li>
<li>Jersey</li>
<li>Json</li>
<li> Html - Bootstap Studio </li>
<li>javaScript - jQuery - AngularJS</li>
<li>CSS</li>
</ul>
